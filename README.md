# Dashboard de Vendas Xbox Game Pass - Desafio DIO

## 📝 Sobre o Projeto
Este repositório contém a entrega do desafio de projeto focado em **Excel para Análise de Dados**. O objetivo principal foi transformar dados brutos de uma base de assinantes do Xbox Game Pass em um Dashboard estratégico, organizado e visualmente atraente para facilitar a tomada de decisão.

## 📊 Perguntas de Negócio Respondidas
O dashboard foi construído para sanar dúvidas críticas sobre o desempenho dos planos, com foco especial em assinaturas **Anuais**:

1.  **Receita de Planos Anuais:** Qual o faturamento total gerado apenas pelos planos de ciclo anual? (Total: 1.754)
2.  **Impacto da Auto Renovação:** Qual o faturamento de planos anuais segmentado por usuários que utilizam ou não a renovação automática? (Sim: 1.537 | Não: 217)
3.  **Vendas EA Play:** Qual o faturamento gerado pelo adicional EA Play dentro da categoria Ultimate? (Total: 600)
4.  **Vendas Minecraft Season Pass:** Qual o volume financeiro do passe de temporada de Minecraft nos planos Standard e Ultimate? (Total: 940)

## 🛠️ Tecnologias e Metodologia
Para garantir a organização e escalabilidade do arquivo, utilizei a estrutura de camadas:
* **Bases:** Dados brutos sem tratamento.
* **Cálculos:** Tabelas dinâmicas e fórmulas para processamento das métricas.
* **Assets:** Identidade visual baseada nas cores oficiais do Xbox (#9BC848, #22C55E).
* **Dashboard:** Camada visual final com gráficos e segmentadores de dados.
